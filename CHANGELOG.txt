Versão 1.1 - Simplificação da Interface
----------------------------------------
Data: 2025-10-19

MUDANÇAS:
- ✓ Removido botão "Ver Craft" (modal) da página Home
- ✓ Mantido apenas botão "Ver Detalhes" que já funcionava perfeitamente
- ✓ Simplificação do código JavaScript (removido modal.js)
- ✓ Removido carregamento de partials/modal.html
- ✓ Interface mais limpa e direta
- ✓ Melhor experiência do usuário - direto para a página de detalhes

MOTIVO:
O modal apresentava problemas de carregamento dos elementos. Como a página
de detalhes já exibe todas as informações necessárias (grade 3x3, ingredientes
e descrição do item), decidimos manter apenas ela, simplificando a navegação.

ARQUIVOS MODIFICADOS:
- js/home.js (simplificado, removido código do modal)
- home/index.html (removido container do modal)
- README.txt (atualizada documentação)

ARQUIVOS NÃO UTILIZADOS (mas mantidos no projeto):
- js/modal.js
- partials/modal.html
